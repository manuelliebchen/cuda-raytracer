# Cuda Raycaster
 A simple cuda enabled raycaster that was created in tearms of a lecture in 2017.

# Build
```
git clone https://github.com/manuelliebchen/cuda-raytracer
cd cuda-raytracer
mkdir build
cd build
cmake ..
make -j
```

# Usage
Use like:
  `$ ./raycaster [Scene File] [Image Folder]`
example:
  `$ ./raycaster scene/scene.sce image/`
